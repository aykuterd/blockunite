How to Play

  1. Drag numbered blocks from the selection area onto the 8x8 grid
  2. Match three or more identical blocks together to merge them
  3. Create higher-value blocks and reach your goal
  4. Plan strategically - every placement counts!
  5. Clear space by creating smart merges
